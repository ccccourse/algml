# OCR

## 安裝

先下載 tesseract 語音辨識執行檔安裝之

* https://github.com/UB-Mannheim/tesseract/wiki

windows 會安裝在 C:\Program Files\Tesseract-OCR

然後再安裝 python 套件

```
$ pip install pytesseract
```

## 執行結果

```
$ python ocr1.py
just do it you can make it

你 不 需要 很 厲害 才 可 以 開 始
你 需要 開 始 才 會 很 厲害
```
